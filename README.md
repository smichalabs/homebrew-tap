# homebrew-tap

Homebrew tap for [smichalabs](https://github.com/smichalabs) tools.

## Usage

```bash
brew tap smichalabs/tap
```

Then install any tool:

```bash
brew install smichalabs/tap/bctl
```

## Available formulae

| Formula | Description |
|---------|-------------|
| `bctl` | A polished CLI for Britive JIT access management |
